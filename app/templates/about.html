What?

{{SITE_NAME}} is a website created to aid UofC students in their program course planning. It has a Course Browser that allows users to quickly look through the university's full course catalog, with intuitive filter and sorting options. Each course has an individual page with all the essential information for students, and a comment section for course-related discussions. With an optional account, users can save courses with custom tags and add them to a virtual Course Planner, where they can be arranged into their respective terms. The Planner will show an interactive GPA calculator and the current progress of the student.
{{SITE_NAME}} also offers a public Web API to easily retrieve Grade, Term, Faculty, Subject and Course data within your own application. The available endpoints are listed in the API documentation.
We plan to regularly add many more features that will keep on improving students' experience making the best choices for their career plan.

Why?

{{SITE_NAME}} was inspired by the need for a better system for browsing and looking for courses. The University's official medium for searching for course information, the Calendar, has been practically left untouched since 2009 (according to the Wayback Machine). In terms of functionality, the official Calendar makes it hard to sort through the selection of courses available without any filtering functionality. Even with a good course browsing experience, students are left on their own to select their desired courses and plan them in a coherent way. {{SITE_NAME}} aims to provide a uniform experience for general course planning that adapts to every student, with more features to come.

How?

The back-end of this website was developed in Python using the Flask micro web framework. To store all the site data, including user data, a relational MySQL database is used and connected to Flask using the SQLAlchemy library. User-sensitive data, like passwords, are stored as a hash using the Bcrypt hashing algorithm. Front-end-wise, the layout and design of this website was created using the Bootstrap framework. The website is currently hosted in PythonAnywhere.
